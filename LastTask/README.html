index.html

  • MEMBER,MUSIC,HISTORYのそれぞれを押すと、対応した紹介ページに遷移します
  • SNSアイコンをクリックするとそれぞれのSNSページに遷移します

    • member.html

  • articleタグではそれぞれのメンバーの紹介を行っています。画像をクリックするとページトップに戻ります
  • asideタグではメンバー名、楽曲名、歴史年を記載しています。メンバー名をクリックするとそれぞれの名前に対応した項目に遷移します

    • music.html

  • articleタグでは一番有名な曲、個人的に聞いてほしい曲、最新曲の順で掲載しています。画像をクリックするとその曲のMVに遷移します
  • asideタグではメンバー名、楽曲名、歴史年を記載しています。楽曲名をクリックするとその曲に対応したMVに遷移します。

    • history.html

  • articleタグではMrs.GreenApple結成から今日に至るまでの主な歴史を掲載しています。
  • asideタグではメンバー名、楽曲名、歴史年を記載しています。年を入力すると、その年に合った項目に遷移します
